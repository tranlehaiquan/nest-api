# NestJS + PrismJS

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ docker compose up -d
$ cp .env.example .env
$ npx nx run-many --all --target=dev
```

## Test

```bash
# unit tests
$ pnpm run test
```
